Extra Notes
===========


Minitel used: Minitel 2, manufactured by Philips

To put the minitel in terminal mode, press `Fnct`-`T` followed by `V`.


# Setup

- `python -m venv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`

Connect your minitel and test with

- `python max-test.py`
